Question1:What is DOM and what is the purpose of DOM?

Answer:

DOM full meaning is Document Object Model. It is a programming interface for HTML documents. It is a logical structure of documents and how a document is accessed and manipulated. The purpose of DOM is dynamically access and update the content, structure, and style of documents. It also provides a way to represent a document's logical structure in a way that is independent from the actual presentation of the document on a particular device.

Question2:How will you select HTML elements using DOM? Name the DOM methods.

Answer:

Some Method that we use for Select Elements:
1. getElementById(): The getElementById() function is used to choose an element based on its id.
2. getElementByClassName(): is a method used to get a collection of elements with the specified class name. It returns an array-like object of all child elements which have all of the given class names.
3. getElementsByTagName() : The getElementsByTagName() function selects all items that have a given tag name.
3. querySelector(): Using this method, you may choose the first element that a given CSS selector matches.
4. querySelectorAll(): With this method, all items that match a given CSS selector are chosen.

Question4:What is a callback function and why will you use it?

Answer:

callback function:
A callback function is a function that is run after the outer function has finished running and is passed as an input to another function. Callback functions, which allow code to be executed after a certain job has been finished, are commonly used in asynchronous programming. By allow the code to run in the background instead of needing to wait for a specific job to complete, this can enhance a program's overall performance.
